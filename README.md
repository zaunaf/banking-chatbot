# Banking Chatbot

Chatbot demo untuk query database SQLite lokal menggunakan Gemini, LangChain, LangGraph, dan Streamlit.

## Fitur

- Login anggota dengan No HP dan PIN
- Cek saldo simpanan anggota
- Melihat maksimal 5 transaksi simpanan terakhir
- Melihat daftar produk simpanan
- Menjelaskan produk simpanan satu per satu
- Melihat daftar produk pembiayaan
- Membuat simulasi pembiayaan dalam tabel markdown

## Tech Stack

- Python
- Streamlit
- SQLite
- LangChain
- LangGraph
- langchain-google-genai
- Gemini API
